# Abstract

This will contain all the details on how the webapp should look. From tech stack, to how data will flow, and the forms we are gonna need.

# Tech stack (packages)

- React
- NextJS
- Zod
- react-hook-form
- React Query
- TailwindCSS
- Shadcn (for the fields and basic components)

# First phase

We will build a basic login form using Shadcn and TailwindCSS. While it is basic, it needs to be modern and simple. No extra fluff, but clean looking.

Should have a basic API integration using React Query to manage the fetches and dispatching mutations. I don’t need a reducer at the moment. A Context for Auth should suffice to hold the current user state.
